A clustering study using ML algorithms with R programming language in retailing

## Guide
Before use these codes, you will need datasets csv file which is named `veri_segm.csv`. Variables are needed is as follow;
* bill_amount	
* bill_amount_avg	
* geo_reg_code	
* location_code	
* income_status	
* store_area	
* stock_tr	
* GMROI	
* stock_goods
